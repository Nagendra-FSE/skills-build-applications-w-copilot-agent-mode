# OctoFit Tracker Frontend

This frontend should be created with Create React App inside `octofit-tracker/frontend`.

## Recommended setup

```bash
npx create-react-app /workspaces/skills-build-applications-w-copilot-agent-mode/octofit-tracker/frontend --template cra-template --use-npm
npm install bootstrap --prefix /workspaces/skills-build-applications-w-copilot-agent-mode/octofit-tracker/frontend
npm install react-router-dom --prefix /workspaces/skills-build-applications-w-copilot-agent-mode/octofit-tracker/frontend
```

Then add the Bootstrap import at the top of `src/index.js`:

```js
import 'bootstrap/dist/css/bootstrap.min.css';
```
